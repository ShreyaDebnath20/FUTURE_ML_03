### Exploratory Data Analysis Findings

- Dataset contains resume information for multiple job roles.
- No significant missing values were found.
- Job roles are fairly balanced across categories.
- Recruiter decisions show the distribution of selected and rejected candidates.
- AI scores range between low and high values, indicating varying candidate quality.
- Experience and project count appear to influence recruiter decisions.
- Numerical features such as AI Score, Experience, and Projects Count may be useful for prediction.
